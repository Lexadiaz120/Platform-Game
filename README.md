# WaterGirl

 ## Try the demo  :  https://lexadiaz120.github.io/Platform-Game/

- Canvas

## Features 
 - Use Arrow Keys to move the player  
 - Gain all diamonds 
 - Avoid obstacules 
 - Soon we add new player 
 - You have 60 second to reach the door and win the game if no you'l die
